# Riot Games API Challenge 2015 
https://www.riotgames.com/en/news/riot-games-api-goals-and-design

<p>As a (former) League of Legends Teemo player, I was always wandering how to build my character in the most damage-effective way I can, as he can go Magic or Physical.
The goal of this (unfinished) project during the challenge was to calculate the DPS for a given skill in a certain amount of time with different sets of items.</p>
